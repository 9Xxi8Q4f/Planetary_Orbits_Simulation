# A 2D Planetary orbit movement simulation with real world values and phsycs

Inspired from **https://fiftyexamples.readthedocs.io/en/latest/gravity.html**
Based on the  
``F = G\frac{m_1m_2}{d^2}`` formula

To run: ``python -m loop.py``

Requirements
--------
Python => 3.11
pygame
